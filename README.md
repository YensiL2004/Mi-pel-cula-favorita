# Culpa Mía - Sitio Web de Personajes

Sitio web informativo sobre la película "Culpa Mía", creado como proyecto de programación web. La página presenta los personajes principales, descripciones individuales y un formulario de registro con estilo responsivo usando Bootstrap.

---

## Tecnologías utilizadas

- ![HTML5](https://img.shields.io/badge/HTML5-E34F26?style=flat-square&logo=html5&logoColor=white) `HTML5`
- ![CSS3](https://img.shields.io/badge/CSS3-1572B6?style=flat-square&logo=css3&logoColor=white) `CSS3`
- ![Bootstrap](https://img.shields.io/badge/Bootstrap-7952B3?style=flat-square&logo=bootstrap&logoColor=white) `Bootstrap 5`

---

## Estructura del proyecto

- `index.html` - Página principal del sitio.
- `css/bootstrap.min.css` - Biblioteca de estilos de Bootstrap.
- `css/style.css` - Estilos personalizados del proyecto.
- `img/` - Imágenes de personajes y portada.
- `Pages/` - Páginas individuales de personajes y registro.
  - `Pages/Noah.html`
  - `Pages/Nick Leister.html`
  - `Pages/Rafaela.html`
  - `Pages/Jenna.html`
  - `Pages/Registro.html`

---

## Descripción del sitio

Este proyecto presenta una página web temática dedicada a la película "Culpa Mía". Está diseñada para mostrar:

- Información general de la película.
- Detalles de personajes importantes.
- Secciones individuales para cada personaje.
- Un formulario de registro sencillo.
- Navegación clara y consistente en todas las páginas.

## Uso

1. Abrir `index.html` en un navegador web.
2. Navegar a través de las páginas de personajes.
3. Usar el formulario de registro para simular el envío de datos.

---

## Notas

- El sitio está construido con HTML estático y CSS.
- Se utiliza Bootstrap para una experiencia responsiva y consistente.
- El diseño se puede ampliar fácilmente a nuevos personajes o secciones adicionales.
